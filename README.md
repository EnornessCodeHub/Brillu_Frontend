Frontend - Enhanced MVP

Features:
  - GrapesJS drag-drop editor (grapesjs + grapesjs-preset-newsletter)
  - Register / Login demo to create tenant + JWT auth
  - Generate MJML template via AI endpoint and render preview in GrapesJS
  - Save campaign (sends MJML to backend for conversion)
  - Configure Mailchimp keys for tenant in settings
  - Send campaign using Mailchimp (tenant must have correct keys and list id)

Setup:
  cd frontend
  npm install
  npm start
